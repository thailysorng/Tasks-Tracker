# Tasks-Tracker
A simple task-tracker with a to-do list using javascript, small html and css

## Features
- **Add tasks**: Type inside input field then press 'Enter' or click on the 'Enter-Icon'
- **Mark tasks as complete**: Click on the box on the left side of task
- **Data storage**: Using local storage to save tasks
- **Delete tasks**: On the right-side of task you can see a trash can icon, press on the icon to remove task
- **Dynamic organize**: Completed task automatically move to bottom, while uncompleted task stay on top
